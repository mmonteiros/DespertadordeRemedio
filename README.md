# Getting Started

1. Clone and Install<p>
  git clone https://github.com/YodaDevs/DespertadordeRemedio.git

2. Navigate to clonned folder and Install dependencies<p>
  cd DespertadordeRemedio && npm install

# Open in your simulator
  	npm run android
